# CSS Box Model 
- The mechanism used by browsers to construct a web page from HTML/CSS code. 

## Vocabulary
- Box Model 
  - box properties
    - width and height
    - padding and margins
    - borders
  
  - visual display models
    - block
    - inline
    - inline-block

  - box sizing model
    - content box
    - border box
  
- Dimensions
  - absolute units
  - relative units 

- General Familiarity
  - container
  - physical pixels
  - CSS reference pixels
